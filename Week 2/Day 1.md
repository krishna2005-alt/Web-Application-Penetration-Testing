## Login Bypass via SQL Injection

## Severity: Critical

## Description
- A SQL Injection vulnerability was identified in the authentication mechanism. Improper handling of user-supplied input allowed authentication controls to be bypassed.

## OWASP Category
- A03:2021 – Injection

## Evidence
- Burp Suite Request Analysis
- Authentication Response Review

## Status
- Verified and Documented
