## Objective
- Install and configure Burp Suite Professional to intercept and analyze web application traffic during security testing.

## Overview
- Burp Suite is a widely used web application security testing tool developed by PortSwigger. It provides features for intercepting HTTP/HTTPS traffic, analyzing requests and responses, and identifying potential security issues in web applications.

## Activities Performed
- Downloaded Burp Suite Professional.
- Installed Burp Suite on the local system.
- Created a new Burp project.
- Configured browser proxy settings.
- Verified successful interception of web traffic.

## Explored key Burp Suite modules including:
- Proxy
- Target
- Repeater
- Intruder
- Logger
- Configuration Steps
- Installed Burp Suite Professional.
- Launched a temporary project.

## Configured browser proxy to:
- IP Address: 127.0.0.1
- Port: 8080
- Enabled Intercept in Burp Proxy.
- Verified that requests from the browser appeared in HTTP History.
- Tools Used
- Burp Suite
- Google Chrome
- OWASP Juice Shop
- Outcome

- Successfully installed and configured Burp Suite. Verified traffic interception and gained familiarity with the interface and core modules required for future penetration testing activities.

## Learning Outcome
- Developed an understanding of how Burp Suite captures and analyzes web traffic, providing a foundation for reconnaissance and vulnerability assessment.
